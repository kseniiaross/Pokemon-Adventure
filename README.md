#  Pokémon Adventure

## Overview
Welcome to PokeAdventure, an interactive web application that allows you to explore the world of Pokémon. This application provides detailed information about your favorite Pokémon, including their abilities, types, and evolutionary paths. Additionally, you can simulate battles to see how your Pokémon perform in combat.


## Features

- **Home Page**: Introduction to the application with a brief overview and a welcoming message.

![Home Page](images/homepage.jpg)


- **Search Page**: Search for any Pokémon by name to get detailed information including stats, abilities, and type.

![Search](images/search1.jpg)
![Search](images/search2.jpg)


- **Pokémon Battle**: Simulation Pokémon battles.

![Battle](images/battle1.jpg)
  


## Technologies Used

- **HTML5** – semantic structure and accessible layout for a responsive interface.  
- **CSS3 / Custom Styling** – refined visual design, animations, and responsive adaptability across devices.  
- **Bootstrap 4.5** – efficient layout management and pre-built components to accelerate UI development.  
- **JavaScript (ES6)** – dynamic interactions, DOM manipulation, and event-driven features for a seamless user experience.  
- **PokeAPI** – real-time Pokémon data integration, including stats, abilities, and evolutionary paths.  
- **Vercel** – modern deployment platform ensuring fast, reliable, and secure hosting.
